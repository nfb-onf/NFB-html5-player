Installing videojs player and nfb skin

Required files

videojs/video.js
The video js core
videojs/videojs.css
The basic css
videojs/nfb/nfb.css
The css for the nfb skin

Html code

<div class="video-js-box nfb-css">
	<video id="video-js" class="video-js" width="515" height="288" controls="controls" preload="auto" poster="poster.jpg">
		<source src="film.mp4" type='video/mp4; codecs="avc1.42E01E, mp4a.40.2"' />

		<embed class="vjs-flash-fallback" src="flash-player.swf" width="516" height="337" allowfullscreen="true" allowscriptaccess="always"></embed>
	</video>

	<p class="vjs-no-video"><strong>Download Video:</strong>
		<a href="film.mp4">MP4</a><br>
		<a href="http://videojs.com">HTML5 Video Player</a> by VideoJS
	</p>
</div>

Javascript code to initializate

VideoJS.setup("video-js", {
	controlsBelow: true,
	controlsAtStart: true,
	controlsHiding: false
});

Capturing and sending user events

Required files

jquery
javascript framework
jquery.video.js
A jquery plugin to manage video events
jquery.video.stats.js
A jquery.video plugin to manage and send nfb events

Javascript code to initializate

You have to use the $.video plugin and adds the nfbStats channel with the film slug and the api key:

$.video('#video-js').addChannel('nfbStats', {
	film_slug: '34813',
	api_key: 'f110c5d6-81c7-55ae-ac82-eeae3ab65d04'
});
